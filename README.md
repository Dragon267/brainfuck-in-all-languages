# brainfuck-in-all-languages 🤌

This is a repo where i write the brainfuck compiler in all programming languages.

## 🐍 Python

```  
python3.11 Python/bf.py main.bf
```

## 📠 C++

```
g++ C++/bf.cpp -o C++/bf
./bf main.bf
```

## 🌐 Javascript

```
node JavaScript/bf.js main.bf
```

## 🚀 Go

```
go build -o Go/bf Go/bf.go
./Go/bf main.bf
```
